Project Overview

  • 5 gestures + idle class → 7-class classifier
  • Input: MediaPipe upper-body pose (15 keypoints × 3 axes)
  • Sliding window (18 frames) → 1 620-dim feature vector
  • NumPy-only neural network (no sklearn / TF / PyTorch)
  • 3 participants — 70/15/15 train/val/test split

Data Pipeline

Three preprocessing steps applied before training

👇 Swipe Down for details

Step 1 — Chest-Centred Normalisation

Translate all keypoints so shoulder midpoint = origin, scale by shoulder width → person-size invariant

Step 2 — Feature Selection

Select 15 upper-body parts (shoulders, elbows, wrists, hands, hips, nose) × 3 axes = 45 position features

Step 3 — Velocity Features

Add frame-to-frame deltas for each position feature → 90 features/frame

Step 4 — Sliding Window

18-frame window (0.6 s), stride 4 → flattened to 1 620-dim input vector

Neural Network Architecture


  Input (1620)
      │
  Dense(256, ReLU) + Dropout(0.3) + L2(1e-4)
      │
  Dense(128, ReLU) + Dropout(0.3) + L2(1e-4)
      │
  Dense(7, Softmax)

  Optimiser : Adam  lr=0.001
  Patience  : 40 epochs   |   LR schedule: ×0.5 / 15 epochs
  Weight init: × 0.1
